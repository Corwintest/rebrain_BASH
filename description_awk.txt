Объяснение кода:
BEGIN { FS=":" }: Определяем разделитель полей как ":", чтобы корректно работать с файлом /etc/passwd.
$3 > 500: Проверяем третье поле (UID), выводим строки, у которых значение UID больше 500.
print $1: Выводим первое поле, то есть имя пользователя (username).

Выполнение команды:
Чтобы запустить скрипт, выполните следующую команду:
awk -f /tmp/parse.awk /etc/passwd
